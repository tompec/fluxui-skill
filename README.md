# Flux UI Skill

An [Agent Skills](https://agentskills.io/) compatible skill for [Flux UI](https://fluxui.dev/) - the official Livewire component library.

## What's included

- `SKILL.md` - Main skill file with component index
- `references/components/` - 44 component reference files
- `references/docs/` - 8 documentation files
- `references/layouts/` - 2 layout files

Each component reference includes:
- YAML frontmatter with `pro` status and `components_used`
- Full documentation and props
- Code examples

## Usage

Copy the repo into your agent's skills directory.
